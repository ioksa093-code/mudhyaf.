

## Description
Mudyaf is a tourism and cultural platform designed to showcase the beauty, heritage, and attractions of the Hail region. The platform enables visitors to explore historical landmarks, cultural destinations, local events, and tourism services through an intuitive and user-friendly interface, creating a seamless experience for discovering Hail.

## Features
- Explore tourist attractions in Hail.
- Discover historical and cultural landmarks.
- View local events and activities.
- Access tourism-related information.
- Easy navigation and search functionality.
- Responsive and user-friendly design.

## Technologies Used
- Laravel
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap


## Vision
To promote tourism in Hail and enhance visitors' experiences by providing easy access to information about attractions, culture, heritage, and local activities through a modern digital platform.
